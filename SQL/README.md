## SQL coding test 준비 (MYSQL)

### 조회(SELECT)
- SELECT FROM

### 정렬 ORDER BY
- 오름차 순: `ORDER BY 기준열`
- 내림차 순: `ORDER BY 기준열 DESC`
- 다중 정렬: `ORDER BY 필드1, 필드2`

### 조건절 WHERE
- 하나의 기준열만 비교: `WHERE 기준열=비교하고 싶은 것`
- 해당 조건이 아닌 경우: `WHERE NOT 기준열=비교하고 싶은 것`
- 문자열 검색: `WHERE 기준열 LIKE "*문자열*"`
- 조건절
    - =, <=, >=
    - and, or, NOT
